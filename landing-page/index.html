常见问题

Q: 数据安全吗？

A: 非常安全。OpenClaw 运行在你的本地电脑上，所有对话数据都在本地处理和存储。中转服务仅负责消息转发，不存储任何对话内容。

Q: 需要公网 IP 吗？

A: 不需要。插件使用 WebSocket 长连接穿透 NAT，无论你在家里、公司还是咖啡厅，只要有网络就能使用。

Q: 支持哪些操作系统？

A: 支持 macOS、Linux 和 Windows (通过 WSL)。需要 Python 3.8 或更高版本。

Q: 如何切换云端和本地实例？

A: 在微信中发送 /switchclaw 命令，即可在云端 OpenClaw 和本地 OpenClaw 之间切换。

Q: 遇到问题怎么办？

A: 可以在 GitHub Issues 提交问题，或发送邮件至 services@7color.vip。
